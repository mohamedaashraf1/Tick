# Tick

Minimal. Focused. Elegant.

A premium dark-themed productivity app built with vanilla JavaScript, featuring animated shader backgrounds, smart task management, and a clean modern UI.


</div>

---

## ✨ Features

- Minimal premium UI
- Animated shader background
- Task priorities
- Due dates & overdue detection
- Search & filtering
- Bulk task actions
- Progress tracking
- Keyboard shortcuts
- Inline task editing
- Duplicate task prevention
- Local storage persistence
- Fully responsive layout
- Smooth animations & transitions

---

## 🧠 Built With

- HTML5
- CSS3
- JavaScript
- Canvas API
- LocalStorage API

---

## ⌨️ Keyboard Shortcuts

| Key | Action |
|---|---|
| `N` | Focus task input |

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/mohamedaashraf1/Tick.git
```

Open the project folder:

```bash
cd Tick
```

Run with Live Server or simply open:

```bash
index.html
```

## 🎨 Design Philosophy

TICK was designed to feel calm, modern, and distraction-free.

The interface focuses on:
- clean typography
- soft motion
- subtle shader visuals
- minimal interaction noise
- productivity without clutter
